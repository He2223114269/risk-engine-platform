# 📋 日常工作 & 计划记录体系

## 文件结构

```
workspace/
├── PLANS.md           ← 当前工作计划（主入口）
├── tasks/
│   └── YYYY-MM-DD.md  ← 每日详细任务/日志
├── memory/
│   ├── YYYY-MM-DD.md  ← 按日归档工作内容
│   └── MEMORY.md      ← 长期记忆
└── 日常工作流程.txt    ← 体系说明
```

## 工作流程

1. **每天开始** → 查看 `PLANS.md` 了解当日计划
2. **执行任务** → 逐条处理，完成后标记 ✅
3. **每个任务完成** → 自动写入 `memory/YYYY-MM-DD.md`
4. **每天结束** → 汇总当天工作到 `PLANS.md` 归档
