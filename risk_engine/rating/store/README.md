# 门店综合评价体系

> 版本: v1.0 | 所属: `risk_engine/rating/store/`

---

## 一、业务背景

门店是代理商的下一级经营实体。门店评级服务于：
- 渠道管控（优质门店激励、风险门店熔断）
- 代理商管理（劣质门店溯源到代理商）
- 风险预警（提前识别正在变差的门店）

## 二、评级维度与权重

| 维度 | 权重 | 说明 |
|:----|:---:|:-----|
| **资产质量** | 30% | 逾期率 + 退订率，最硬指标 |
| **客群质量** | 20% | 新老客、单卡融合、本异网、风控通过率 |
| **渠道关联** | 15% | v3_store 渠道等级 + 代理商评级 |
| **经营基础** | 15% | 活跃月数 + 交易量百分位 |
| **规模趋势** | 10% | 交易额增速 + 逾期趋势 |
| **经营健康度** | 10% | 最近活跃度等 |

## 三、特殊逻辑

- **拉黑渠道** → 直接评为 C 级（硬约束）
- **数据不足的门店**（交易<20笔或活跃<2月）→ 逾期率不给满分，默认 50 分

## 四、数据来源

- `dws.dws_credit_yzf_order_complete` — 各维度聚合
- `ods.ods_ts_v3_order_store` — 渠道等级
- `risk_control.supplier_evaluation` — 所属代理商评级

## 五、运行方式

```python
from risk_engine.rating.store.run import run_store_rating
df = run_store_rating(data_date="2026-05-25")
```
