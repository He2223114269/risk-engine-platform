# 代理商综合评级体系说明

> 版本: v3.0 | 2026年5月

---

## 一、评级目的

对合作代理商进行综合质量评级（A/B/C三级），评级结果用于：

- **差异化保障金缴纳比例**：评级越高，缴纳比例越低
- **差异化业务额度**：评级越高，每月可办单数越多
- **风险预警**：评级下降的代理商及时关注，防止风险扩大

实现 **"好代理商少交钱多办事，差代理商严管控控风险"**。

---

## 二、评级框架

评级综合考量 **三大维度**：

```
┌─────────────────────────────────────────────┐
│              代理商综合评级                    │
├──────────┬──────────┬──────────────────────┤
│  资产质量  │  经营能力  │    企业资质           │
│  (权重40%) │ (权重38%) │    (权重22%)          │
├──────────┼──────────┼──────────────────────┤
│ • 订单逾期率│ • 客群结构│ • 企业正规度（新）      │
│           │ • 业务规模│ • 资本实力（新）        │
│           │ • 展业稳定性│ • 翼支付评级          │
│           │ • 通过率表现│                      │
└──────────┴──────────┴──────────────────────┘
```

每个维度评分 0-100 分，按权重加权汇总得出综合评分，再按排名划分等级。

---

## 三、各维度评分逻辑

### 维度一：资产质量（权重 40%）

衡量代理商的订单逾期情况，**最核心的指标**。

| 逾期率 | 得分 | 说明 |
|:-----|:---:|:----|
| 0%（有足够业务量） | 100 | 真实表现优秀 |
| 0%（业务量不足） | 50 | 还未到还款期，待观察 |
| 2% | 80 | 良好 |
| 5% | 50 | 一般 |
| 10%以上 | 0 | 高风险 |

**关键原则**：业务量不足的代理商不给满分，从严把控。

---

### 维度二：经营能力（权重 38%）

综合评估代理商的业务质量和运营稳定性，包含四个子维度：

#### ① 客群结构（13%）
- **老客户占比越高越好** — 老客户多的代理商服务质量和口碑更好
- **融合套餐占比越高越好** — 融合套餐客户黏性更高、违约率更低
- **本网占比越高越好** — 本网用户稳定性优于异网用户

#### ② 业务规模（11%）
按交易笔数在所有代理商中的排名计分。交易量越大，说明市场份额和获客能力越强。

#### ③ 展业稳定性（9%）
- **活跃时长**：持续开展业务的时间越长越好
- **近期活跃度**：近期是否还在正常展业（60天以上未活跃会扣分）

#### ④ 通过率表现（5%）
- 如果风控通过率高于全省平均水平 → 加分（说明客户质量筛选严格）
- 如果低于平均水平 → 扣分（说明准入门槛偏松）

---

### 维度三：企业资质（权重 22%）

评估代理商本身的企业质量，**这是本次升级新增的重点维度**。

#### ① 翼支付评级（11%）
参考翼支付官方对代理商的 A~E 评级映射得分。

#### ② 企业正规度（6%）（新增）
基于企业工商信息评估：

| 条件 | 影响 |
|:----|:----:|
| 企业登记状态为"存续" | 加分 |
| 企业已"注销"或"吊销" | **明显扣分** |
| 有限责任公司 | 加分（正规经营主体） |
| 个体工商户 | 保持中性 |

#### ③ 资本实力（5%）（新增）
按注册资本规模评分，**注册资本越高抗风险能力越强**。

| 注册资本 | 得分 |
|:--------|:---:|
| 10万元以下 | 40分 |
| 50万元 | 60分 |
| 200万元 | 80分 |
| 1000万元以上 | 100分 |

---

## 四、数据缺失处理

**关键原则：不缺数据不放松，缺数据不扣分。**

| 情况 | 处理方式 |
|:----|:---------|
| **翼支付评级缺失** | 用逾期表现估算评级，表现好默认 B 级，差默认 D 级 |
| **企查查数据缺失** | 该维度权重自动分配给其他维度，总分不受影响 |
| **业务量不足** | 逾期率即使为0也不给满分，标记为待观察 |

**效果验证**：有企查查数据的代理商平均分 73.1，无数据的平均分 71.8，**差异仅1.3分**，说明数据缺失不会不公平地影响评分。

---

## 五、等级划分

| 等级 | 占比 | 说明 |
|:---:|:---:|:------|
| **A级** | **前1%** | 综合表现最优秀的头部代理商 |
| **B级** | 1%~20% | 表现良好的代理商 |
| **C级** | 剩下80% | 需要关注或改进的代理商 |

按评分排名分配，不设绝对分数线，随整体质量动态调整。

## 六、版本对比

| 项目 | 原版本 | 新版本（v3.0） | 变化 |
|:----|:-----:|:-------------:|:----|
| 评估维度 | 6个 | **8个** | +2个企查查维度 |
| 数据来源 | 业务数据 | 业务数据 + **企查查工商数据** | 信息来源更全面 |
| 正规度评估 | 无 | 有（存续/注销判别） | 能识别已注销风险代理商 |
| 资本实力评估 | 无 | 有（注册资本） | 能区分不同规模代理商 |
| 数据缺失处理 | 不处理 | **动态权重调整** | 缺数据不被扣分 |
